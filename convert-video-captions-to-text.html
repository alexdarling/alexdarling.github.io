How to use this tool

Open your SubRip file in a text editor of choice. Copy the SubRip file into the text area above and click "Convert to Text". If you want the first line to be a title (like in the example below), check the box.

What is this tool used for?

Websites that offer captioning services like Wistia will give you captions as a SubRip file (with the *.srt extension). You can open SubRip files in a text editor, but if you want to turn those into a block of text you will be stuck with something that looks like this:

1
00:00:01,500 --> 00:00:03,750
The Tale of Sir Robin

2
00:00:05,000 --> 00:00:07,750
So each of the knights
went their separate ways.

3
00:00:08,100 --> 00:00:10,750
Sir Robin rode north,
through the dark forest of Ewing...

And so on. Someone could technically read that, but it's an awful user experience and I'd be surprised if it didn't confuse search engines. If we could get it to display like this instead:

The Tale of Sir Robin.

So each of the knights went their separate ways. Sir Robin rode north, through the dark forest of Ewing...

That would be a lot more useful.
